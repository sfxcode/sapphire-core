    Sapphire Extension

    The Sapphire Extension Project adds additional features to sapphire-core.

    It provides some useful helpers for javafx applications build on top of the sapphire-core framework.

    Features

    Common

    • Advanced Table Support
    • Advanced Filter Support

    Controls

    • DataListView
    • DualDataListView

    Controller

    • DataTableController
    • EditorController
    • Maser/Detail Controller
