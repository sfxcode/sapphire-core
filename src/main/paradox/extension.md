# Sapphire Extension

The [Sapphire Extension Framework](https://github.com/sfxcode/sapphire-extension) adds additional features to sapphire-core.

## Features

### Common
- Advanced Table Support
- Advanced Filter Support

### Controls
- DataListView
- DualDataListView

### Controller
- DataTableController
- EditorController
- Maser/Detail Controller
