    sapphire-core

    A JavaFX (ScalaFX) Application Framework for Scala User. It combines scala programming patterns with MVC for building complex JavaFX Applications.

    Additional Documentation

    JavaFX

    Java UI Application Framework as replacement for Swing.

    Sapphire depends on JavaFX 8, which is included in JDK 8.

    http://www.oracle.com/technetwork/java/javafx/overview/index.html

    ScalaFX

    Sapphire depends on ScalaFX, A DSL for JavaFX written in Scala.

    Dependency Injection

    Sapphire use Apache Deltaspike as CDI Abstraction Layer (1.8.x).

    The default CDI implementation depends on Apache OpenWebBeans (2.0.x).

    Expression Language

    Expressions are resolved by JUEL.

    Dependency Setup

    sbt
    libraryDependencies += "com.sfxcode.nosql" %% "sapphire-core" % "1.3.3"
    Maven
    <dependency>
  <groupId>com.sfxcode.nosql</groupId>
  <artifactId>sapphire-core_2.12</artifactId>
  <version>1.3.3</version>
</dependency>
    Gradle
    dependencies {
  compile group: 'com.sfxcode.nosql', name: 'sapphire-core_2.12', version: '1.3.3'
}
